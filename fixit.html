About Project

Scope The “FIXIT” is a GPS-based mobile application which helps people to find the closest HM based on the customer’s current position and other specification like price, work type, and more.
The application should be free to download from either a mobile phone application store. HMs can provide their information using the app itself. This information will act as the bases for the search results displayed to the user.
An SM administers the system and keep the information accurate. The SM can, for instance, verify HMs and manage user information. Furthermore,
the software needs both Internet and GPS connection to fetch and display results. All system information is maintained in a database. The software also interacts with the GPS-Navigator software which is required to be an already installed application on the customer’s mobile phone. By using the GPS-Navigator,
customer can view desired HM on a map and HM can view desired customer location and be navigated to them. The application also has the capability of representing both summary and detailed information about the HMs.
